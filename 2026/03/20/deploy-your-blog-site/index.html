本文仅为博客萌新个人技术记录，对于一些诡异的情况不一定考虑周全

如阅读中途遇到任何问题请勤翻官方文档或者使用大模型

工具

  • Hexo
  • git
  • Node.js

具体安装步骤请参考Hexo安装

注意，文档中所有的命令全部在命令提示符（Windows cmd）中完成

链接VScode

如果有需要的话，直接把根目录下的文件拖到VScode里面，代码管理起来更方便

使用过VScode的同学肯定比较熟悉

部署到github

首先你需要有一个github账号（没有就注册一个，非常简单）

接下来，在github页面打开自己的主页（profile），点击repository，在这里可以新建一个仓库，起一个名字（我的是Hexo）并将可见性调为public（请务必这么做），于是你就有一个自己的代码仓库了，可以将你的博客文件部署到此处

接下来的部署操作参考github一键部署

新的开始

使用hexo创建和发布文章十分简单，使用hexo new draft "<title>"创建新的草稿，在source/_drafts文件夹内会看到自己的草稿。

编辑完草稿后使用hexo public draft "<title>"发布你的文章

现在的你应该拥有自己漂亮的博客网站了，开启你的写作之旅吧！
